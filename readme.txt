WDO Birthdays

--------------------

Displays birthdays for the current day via a widget, users can hide their birthday if they wish to. You can edit the look through the CSS file. For more information, or if you need help with the plugin, or to resquest an update, email us at contact@webdevsonline.com.

--------------------

Requires at least: 3.3.2
Tested up to: 3.3.2

--------------------
